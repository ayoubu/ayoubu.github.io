目标

这一部分的主要目标是对数据集进行 探索性数据分析（EDA），EDA （Exploratory Data Analysis），是指对已有数据在尽量少的先验假设下，通过作图、制表、方程拟合、 计算特征量等手段探索数据的结构和规律的一种数据分析方法。EDA 是一种更加贴合实际情况的分析方法，它强调让数据自身 ”说话“，通过EDA我们可以最真实、直接的观察到数据的结构及特征。

EDA一般步骤

检查数据

  • 是否有缺失值
  • 是否有异常值
  • 是否有重复值
  • 数据分布是否均衡
  • 是否需要抽样
  • 变量是否需要转换
  • 是否需要增加新的特征

使用描述统计量和图表对数据进行描述

  • 连续变量：
    • 统计量：平均值、中位数、众数、最小值、最大值、四分位数、标准差等。
    • 图表：频数分布表、直方图、箱线图等。
  • 无序离散变量：
    • 统计量：各个变量出现的频数和占比
    • 图表：频数分布表、柱状图、条形图、茎叶图、饼图等。
  • 有序离散变量：
    • 统计量：各个变量出现的频数和占比
    • 图表：频数分布表，堆积柱状图，堆积条形图等

考察变量之间的关系

  • 离散变量
  • 连续变量

进行数据处理

EDA的结果

  • 数据的直观表现
  • 发现潜在的结构
  • 提取重要的变量
  • 处理异常数据
  • 检验统计假设
  • 建立初步模型
  • 决定最优因子的设置

本次学习目标：

  • 熟悉数据集，了解数据集，对数据集进行验证来确定所获得数据集可以用于接下来的机器学习或者深度学习使用。
  • 了解变量之间的相互关系与预测值之间的关系。
  • 引导数据科学从业者进行数据处理以及特征工程的步骤,使数据集的结构和特征集让接下来的预测问题更加可靠。
  • 完成对于数据的探索性分析，并对于数据进行一些图表或者文字总结并打卡。

探索性数据分析所用到的库：

  • 数据科学库：
    • pandas
    • numpy
    • scipy : Scipy库是基于Python生态的一款开源数值计算、科学与工程应用的开源软件
  • 可视化库：
    • matplotlib
    • seaborn : seaborn 是在matplotlib基础上进行了更高级的 API 封装，从而使得作图更加容易，seaborn可以视为对matplotlib的一个补充。
