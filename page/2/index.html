什么是json

本文字数： 4k 阅读时长 ≈ 4 分钟

简介

JSON的全称是 ”JavaScript Object Notation“，意思是JavaScript对象表示法，它是一种基于文本，独立于语言的轻量级数据交换格式。XML也是一种数据交换格式，为什么没有选择XML呢？因为XML虽然可以作为跨平台的数据交换格式，但是在JS（JavaScript）中处理XML非常不方便，同时XML标记比数据多，增加了交换产生的流量，而JSON没有附加的任何标记，在JS中可作为对象处理，所以我们更倾向于选择JSON来交换数据。

JSON的两种数据结构

JSON有两种表示结构，对象和数组。对象结构以{ 大括号开始，以}大括号结束。中间的信息以键值对存储，并且以,分割每条信息（最后一条数据不用加,）。键值对的格式为：key:value，即冒号左边为关键字，冒号右边为值。

MongoDB简介

MongoDB是一个基于分布式文件存储的数据库，由C++语言编写，旨在为WEB应用提供可扩展的高性能数据存储解决方案。

MongoDB将数据存储为一个文档，数据结构由键值对(Key : value)组成，MongoDB文档类似于JSON对象，字段值可以包含其它文档，数组及文档数组。

MongoDB服务端可运行在Linux、Windows和Mac OS平台，支持32位和64位应用，默认端口为27017。本文所展示的为Windows平台的安装及使用。

MongoDB支持各种编程语言：Python、Java、C++、PHP、C#等多种语言。

MongoDB的优势

  • 易扩展： NoSQL（非关系型数据库）数据库种类繁多，但是一个共同的特点都是去掉关系型数据库的关系特性。数据之间无关系，这样就非常容易扩展。
  • 大数据量，高性能： NoSQL数据库都具有非常高的读写性能，尤其在大数据量下，同样表现优秀。得益于它的无关系性，数据的结构简单
  • 灵活的数据模型： NoSQL无需事先为要存储的数据建立字段，随时可以存储自定义的数据格式。而在关系型数据库里，增删字段是一件非常麻烦的事情。

缺点： 由于它每条数据都是单独存储的，因此在存储过程中可能会包含大量重复信息，非常占用存储空间。

电子表格打印

快捷键：Ctrl + P — 打印预览

将表格打印到一页纸上： Ctrl + P ，在最后一项中选择将工作表调整为一页，此方法是使用缩放功能。除此之外也可以使用自定义边距功能，设置页边距使得内容可以显示为一页。

打印选中区域： 先选中需要打印的区域，然后按Ctrl + P，在设置中的第一项中选择打印选定区域。

打印网格线： 首先在页面布局 菜单栏中的工作表选项中，勾选打印，再按Ctrl + P 预览的时候，就可以看到网格线了。

在每页都打印表头： 进入页面布局菜单栏，在页面设置区域中，点击打印标题，然后分别设置打印区域 和顶端标题行。注：在打印预览时，最后一项应该设置为将所有列都调整为一页。

打印页码： 进入页面布局菜单栏，在页面设置区域点开扩展的小箭头，选择页眉/页脚 ，在页脚处设置页码的样式，点击确定就好。

不打印错误值： 还是在上述页面设置的对话框中，选择工作表 ，将错误单元格打印为设置为空白就好。

打印批注： 还是在页面布局菜单栏中的页面设置对话框中，选择工作表，设置批注和注释为工作表末尾，此时再打印时，批注就会打印到末尾了。

设置单色打印：此时会自动忽略所有表格内的色彩，在上述的页面设置中的工作表选项卡里，勾选单色打印的复选框即可。

图表与PPT、Word之间的交互联动

在实际工作中，我们经常需要将excel中的结果插入到PPT或者Word中，如何在改动excel表格数据时，PPT和Word中的数据也会相应的改变。从而实现数据的实时同步。

具体方法为： 在复制Excel表格后粘贴到PPT或者Word时，在开始菜单中点击粘贴底下的小箭头，选择选择性粘贴，然后在弹出的对话框中，选择粘贴链接中的Microsoft Excel工作表 对象 即可。

粘贴后Excel的表格样式什么的会粘贴过去。如果Excel、PPT、Word三个文件均为打开状态，此时修改Excel中的数据，其它两个文件中的数据也会发生相应的更改。如果修改Excel数据时其它两个文件时关闭的，那么再次打开其它两个文件时会有提醒是否同步。为了保证数据之间的同步，最好时将Excel数据和引用它的文件放在一起，如果修改了文件名或者目录可能会导致链接不到。

合并多个工作表

新建一个工作表，选中第一个表格，单击菜单栏中的数据选项卡，在数据工具区域中选择合并计算，在引用位置选择第一个表中的数据，然后点击添加，依次添加所有表，最后勾选首行和最左列复选框，点击确定即可。

首行表示显示第一行的标题，最左列表示显示最左列的数据。
